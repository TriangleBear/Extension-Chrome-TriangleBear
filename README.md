# Extension Chrome TriangleBear
Custom Extention made by me

# How to Install
1. Download the Extension from the Chrome Web Store or from the GitHub repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the folder where you downloaded the Extension.
5. The Extension should now be installed and ready to use.

# Extensions

- [Messenger Notifier](messenger-notifier): This extension will notify you when you receive a new message on Facebook Messenger specifically from a selected conversation.
